Usage

jquery.fillintheblanks is a plugin to be used on form elements with two textareas — one with the original text and one with the result text.

Sample code

The code below will create a fill in the blanks generator using the form "frm-fill". The symbol used for the blanks will be an empty space; the size will be 3 times the word length; and there will be 2 blanks by line.

$('#frm-fill').fillintheblanks({
  size: 3,
  symbol: ' ',
  blanks_by_line: 2
});
