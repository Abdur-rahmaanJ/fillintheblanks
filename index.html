Usage

jquery.fillintheblanks is a plugin to be used on form elements with two textareas — one with the original text and one with the result text.

Sample code

The code below will create a fill in the blanks generator using the form "frm-fill". The symbol used for the blanks will be an empty space; the size will be 3 times the word length; and there will be 2 blanks by line. Finally, the words "the", "in", "on" and "as" will be ignored during the process.

$('#frm-fill').fillintheblanks({
  size: 3,
  symbol: ' ',
  blanks_by_line: 2,
  ignore: ['the', 'in', 'on', 'as']
});

Full list of options

Option Default Usage
size 1 Size of the blank mark. This option will be multiplied by the word length.
symbol _ Defines the symbol to be used when a word is erased.
blanks_by_line 1 Number of words to be erased in a line.
min_size 0 Minimum size (in letters) of words to be erased.
max_size 0 Maximum size (in letters) of words to be erased (0 means unlimited).
ignore empty An array containing words that should be ignored when searching for words to erase.
